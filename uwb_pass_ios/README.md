# UWB-Pass — iOS Premium UI

Aplikacja desktopowa w Pythonie/Tkinter z eleganckim interfejsem inspirowanym iOS.

## Uruchomienie w VS Code / PowerShell

```powershell
py -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
py main.py
```

## Funkcje

- konto użytkownika i hasło główne,
- szyfrowany sejf AES-GCM,
- dodawanie loginów i notatek,
- generator haseł,
- audyt zdarzeń,
- eksport CSV,
- jasny interfejs premium w stylu iOS.
